# 🧠 Mental Health Chat UI

This project is a **basic front-end user interface for a mental health chat website**, designed to provide a simple and supportive digital experience for users.

The application begins with an **Emotion Selection Page**, where users can choose how they are feeling. Based on their selection, they are guided to a **Chat Page**, simulating a conversation interface aimed at offering comfort and interaction.

Additionally, the project includes a **Journal Page**, allowing users to express their thoughts and emotions in a more personal way. This flow helps create a structured yet flexible mental wellness experience.

## ✨ Features

* Emotion-based entry point for personalized interaction
* Simple and clean chat interface
* Journal page for self-expression
* Smooth navigation between pages

## 🛠️ Tech Stack

* HTML (structure)
* CSS (styling)
* JavaScript (basic interactivity)

## 📌 Note

This project focuses on **UI design and basic functionality**, and does not include backend or real-time chat features.

---

This project demonstrates how front-end technologies can be used to design meaningful and user-friendly interfaces in the mental health domain.
